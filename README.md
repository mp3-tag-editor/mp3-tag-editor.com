## 🚀 What Is MP3 Tag Editor?

**[MP3 Tag Editor](https://mp3-tag-editor.com)** is a free, browser-based audio metadata editor that lets you edit ID3 tags, Vorbis comments, and iTunes metadata for any audio file — directly in your browser. No desktop software, no account, no cost.

Whether you want to fix song titles, update album artwork, add artist info, or clean up your entire music library, MP3 Tag Editor gets it done in seconds.

### ✨ Key Features

- **Bulk Editing** — Edit up to 5 files simultaneously; apply shared tags or edit each file individually
- **Complete Metadata Support** — Title, artist, album, year, genre, track number, composer, lyrics, and album art
- **Cover Art Management** — Add, replace, or remove album artwork (cover images) from any audio file
- **Format Agnostic** — Supports MP3, WAV, FLAC, OGG, M4A, AAC, and more
- **ID3v1 & ID3v2** — Automatic upgrade to ID3v2 when adding extended data or artwork
- **Privacy First** — Files are processed securely and never stored permanently
- **No Install Needed** — Works entirely in your browser on any OS
- **Android App** — Take tag editing on the go with the [AudioTagger Android app](https://play.google.com/store/apps/details?id=com.almas.audiotagger)

## 🏷️ What Are Audio Tags / ID3 Tags?

Audio tags (also called **ID3 tags**, **metadata**, or **music tags**) are invisible data fields embedded inside audio files. They store information about the track so music players can display it correctly.

### Common Tag Fields

| Field | Description |
|---|---|
| **Title** | The name of the song or track |
| **Artist** | The performer(s) of the music |
| **Album** | The collection or release the track belongs to |
| **Year** | When the music was released |
| **Genre** | Musical style or category |
| **Track Number** | Position within the album |
| **Composer** | Who wrote the music |
| **Album Art** | Embedded cover image |
| **Lyrics** | Song lyrics (ID3v2 only) |

### ID3v1 vs ID3v2

- **ID3v1** — Original format, limited to 128 bytes, basic fields only
- **ID3v2** — Modern, flexible format supporting images, lyrics, and custom fields

MP3 Tag Editor fully supports both versions and automatically upgrades to ID3v2 when needed.

> 💡 **Did you know?** The name "ID3" stands for *IDentify MP3*. The standard was introduced in 1996 to bring metadata capabilities to a format that originally had none.
